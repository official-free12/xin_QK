# 信佣卡还款记账管理工具 记账工具 银行卡记账工具

用户添加自己所有的信佣卡，后续银行卡账户变动操作一下，然后可以实时查看信佣卡余额和待还金额。方便快捷管理。支持公众号端口，也可以后台操作。可参考微信公众号"卡账记"<br>
